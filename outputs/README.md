# Outputs

This directory contains generated output files from analysis runs.

All files in this directory are gitignored and will not be committed to the repository.

## Typical Contents

- `*.png` - Pore size distribution plots
- `*.ply`, `*.obj` - 3D mesh exports of segmented pores
- `*_stats.txt` - Statistical analysis results
- `*_sizes.txt` - Individual pore size data
- `*_report.txt` - Detailed analysis reports
